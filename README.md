# Multi-Agent ReAct AI Assistant

A stateful AI agent system built using LangGraph + Ollama.

## Features
- Multi-agent architecture
- ReAct reasoning
- Tool calling
- DuckDuckGo search
- Safe calculator
- Notes memory
- Therapist agent
- Streamlit UI

## Tech Stack
- Python
- LangGraph
- Ollama
- Streamlit
- DDGS

## Run

```bash
streamlit run app.py
